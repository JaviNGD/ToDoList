## Run Locally 

Clone the project
```plaintext
git clone https://github.com/JaviNGD/ToDoList.git
```

Go to the project directory
```plaintext
cd ToDoList
```

Install dependencies
```plaintext
  npm install
```

Start the server
```plaintext
  npm start
```
